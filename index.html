Hello Anytypers! We’re excited to share a major update you’ve been eagerly awaiting. Version 0.45.0 introduces Web Publishing, a Raycast extension, RTL support, and a host of quality-of-life improvements—all inspired by your requests and feedback. Thank you for helping us shape Anytype into a friendlier, more powerful tool for everyone.
Highlights
Web Publishing
You can now publish Objects as static web pages (HTTPS links) on your personal subdomain if you have *any name. These pages are uploaded to our servers as unencrypted HTML files.
This is an early version—mobile optimization, linked Objects, and blocks like Sets, Collections, and Relations aren’t supported yet. Multi-page publishing and other enhancements are on the way, and we’d love your feedback to guide these improvements.
Redesigned Space Panel & Navigation
We’ve relocated the navigation controls to the Space Panel for a smoother experience. You can now navigate back and forth, search for Objects, and create new ones from a familiar place.
Pro tip: if your sidebar is hidden, press ⌘ + S to open Global Search or ⌘ + N to create a new Object.
Raycast Extension (macOS)
Our long-awaited API is beginning to take shape! The first step is Raycast integration, allowing you to create, read, and delete Spaces, Objects, and Types, as well as search across them, all from Raycast on macOS.
While this initial version focuses on a basic set of features, we’re laying the groundwork for an open API that will eventually support bulk import and export, NoCode tools, and more integrations. We’re excited to see what you’ll build!
Install the Raycast Extension here
RTL Support
We’re happy to introduce right-to-left language support! This first iteration might need some extra polish, so your feedback is invaluable. Let us know what works and what needs a bit more attention.
Quality-of-Life
Add Existing Objects to a Collection
We’ve added an option to add existing Objects into a Collection. Look for it in the updated arrow menu alongside other content settings.
Horizontal Alignment in Grid Layout
Columns in the Grid layout can now be aligned left, center, or right.
Simplified Widget Settings Menu
Appearance options are now limited to changing the widget’s view (Compact List, List, View, Link).
The "Number of Objects" setting has moved to its own section.
Clickable Links in Kroki Diagrams
Kroki embed blocks now support links within diagrams. Thanks to @siousu for the suggestion!
Improved Dark Mode Text Selection
We tweaked the selection color for better visibility. Shout-out to @kermit_frog for flagging this!
Inline Set Header Resize Adjustments
Inline Set headers now adapt when resizing to keep controls from overlapping neighboring blocks.
Better Image Pasting on macOS Safari
Images copied from Safari will now paste into the object editor as files instead of URLs.
Standardized QR Code Appearance
All QR codes in Anytype now follow a consistent style.
Improved LaTeX Support
Inline LaTeX is now rendered in the Table of Contents block and in link blocks, including titles and descriptions. Thanks to @mewald!
